Este script SQL crea la estructura de una base de datos en SQL SERVER para la gestión de bienes raíces, almacenando información sobre provincias, localidades e inquilinos. 
Se incluyen claves primarias, claves foráneas e índices para optimizar las búsquedas.

Cómo Utilizar

Asegúrese de tener un servidor SQL Server en ejecución.

Cree la base de datos ejecutando:

CREATE DATABASE RealState;

Seleccione la base de datos antes de ejecutar el script:

USE RealState;

Ejecute el script Base de datos.sql en un entorno de administración SQL Server como SQL Server Management Studio (SSMS).

Una vez ejecutado, la base de datos estará lista para almacenar información.
